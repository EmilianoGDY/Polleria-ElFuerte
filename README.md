# Pollería "El Fuerte" — Simulación de colas

Trabajo Práctico Integrador de **Simulación** (Ingeniería en Sistemas de
Información, UTN-FRRE).

Simula la atención de una pollería de venta de milanesas con **un solo operario**
(sistema de colas de servidor único, FIFO). Genera las variables aleatorias por
método Montecarlo con distribuciones empíricas tomadas de datos de campo, y
permite analizar el efecto del local con y sin stock pre-elaborado sobre la espera 
y la ganancia.

# Probar la simulación
- Clonar el repositorio con git clone en carpeta deseada.
- Entrar a dicha carpeta en la terminal y correr el siguiente comando 
  "py ModeloInterfaz/web/servidor.py"
para correr el modo interfaz de la simulación (recomendado).
